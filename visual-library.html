为什么内容创作者需要 AI 生产管线

不是 AI 帮你想内容，而是让你从脚本到切图的整个流程提速三倍。

Card · 卡片
✦

Claude Code 是什么

命令行 AI 工具
Anthropic 出的终端 AI 编程工具，不需要打开 IDE，直接在命令行里操作整个代码库。对内容创作者的价值在于：能读懂你的整个项目结构，执行批量文件操作，自动化重复任务。
△

适合什么人用

有一定技术基础、日常重度使用终端的人。纯内容创作者如果不熟悉命令行，上手成本会比较高。
Steps · 编号步骤
1
安装 Node.js 和 Chrome
Node.js 18+ 即可，Chrome 用于 Puppeteer 截图导出，脚本会自动检测路径。
2
克隆工具包并安装依赖
git clone 后在目录内跑 npm install puppeteer-core，约 30 秒完成。
3
运行 skill，完成初始化
告诉 AI「帮我做小红书图文」，首次运行会引导你选配色方案，约 1 分钟搞定。

Pitfall · 常见坑
① 别把 AI 当写手
让 AI 直接生成脚本发布，读者一眼能看出来。AI 的价值在流程自动化，不在内容替代。脚本还是得自己写，个人经历和观点是没法批量生产的东西。

Chat · 对话气泡
🤔
我不懂代码，能用这个工具吗？
💡
可以用，但有门槛。需要会基础的命令行操作和文件路径概念。如果你完全没接触过终端，建议先跟着 README 跑一遍，能跑通就没问题。
🤔
导出图片的尺寸可以改吗？
💡
可以。export-slice.js 的第三个参数是切片高度，默认 1440，改成你需要的数字就行。宽度固定 1080px，符合小红书标准。
